## 📌 Overview
This project implements a machine learning model to classify music into different genres based on extracted audio features. It demonstrates a full data science workflow including preprocessing, feature engineering, model training, and evaluation.

---

## 🛠️ Tech Stack
- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib / Seaborn (optional)

---

## ⚙️ Features
- Data preprocessing and cleaning
- Feature extraction from audio data
- Model training and evaluation
- Genre prediction
- Visualization of results (optional)
